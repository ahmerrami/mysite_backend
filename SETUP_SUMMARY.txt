🔒 SÉCURITÉ - IMPORTANT!

Avant de déployer, vérifier:

1. Générer une nouvelle SECRET_KEY pour la production:
   python -c 'from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())'

2. Vérifier la configuration de sécurité:
   python manage.py check --deploy

3. S'assurer que DEBUG=False en production

4. Configurer les variables d'environnement sur GitHub et PythonAnywhere

5. Activer HTTPS/SSL
